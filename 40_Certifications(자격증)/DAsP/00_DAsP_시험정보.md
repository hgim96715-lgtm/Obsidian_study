

# 한 줄 요약

DAsP는 데이터아키텍처 준전문가 자격으로, 데이터 요건 분석·데이터 표준화·데이터 모델링 중심의 데이터 설계 역량을 확인하는 시험이다.

# 시험 구성

DAsP는 공식 기준으로 총 4과목으로 구성된다.

| 구분 | 내용 |
|---|---|
| 자격명 | 데이터아키텍처 준전문가 |
| 시험명 | DAsP |
| 과목 수 | 4과목 |
| 문제 수 | 객관식 50문항 |
| 시험 시간 | 90분 |

---

# 시험 과목

1. 전사아키텍처 이해
2. 데이터 요건 분석
3. 데이터 표준화
4. 데이터 모델링

---

# 합격 기준

- 총점 60점 이상
- 과목별 40% 미만 과락 없음

즉, 전체 점수가 60점 이상이어도 특정 과목 점수가 너무 낮으면 불합격할 수 있다.

---

# 공부 방향

SQLD에서 배운 SQL과 데이터베이스 기본 개념을 바탕으로, DAsP에서는 데이터가 실제 시스템에서 어떻게 정의되고 설계되는지에 집중한다.

특히 다음 개념을 중심으로 정리하면 좋다.

- 데이터 요건 분석
- 데이터 표준화
- 논리 데이터 모델링
- 물리 데이터 모델링
- 엔터티, 속성, 관계
- 식별자
- 정규화
- 데이터 품질과 표준 관리

---

