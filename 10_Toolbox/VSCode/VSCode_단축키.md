

> Mac 기준 정리  
> Windows는 보통 `Cmd` 대신 `Ctrl`을 사용하면 됨

---

## 자주 쓰는 기본 단축키

| 기능 | Mac 단축키 | 설명 |
|---|---:|---|
| 명령 팔레트 열기 | `Cmd + Shift + P` | VS Code의 모든 명령 검색 |
| 파일 빠르게 열기 | `Cmd + P` | 파일명으로 빠르게 이동 |
| 전체 검색 | `Cmd + Shift + F` | 프로젝트 전체에서 검색 |
| 현재 파일 검색 | `Cmd + F` | 현재 파일 안에서 검색 |
| 바꾸기 | `Cmd + Option + F` | 현재 파일에서 문자열 바꾸기 |
| 사이드바 열기/닫기 | `Cmd + B` | 왼쪽 탐색기 토글 |
| 터미널 열기/닫기 | `Ctrl + \`` | 내장 터미널 토글 |
| 파일 저장 | `Cmd + S` | 현재 파일 저장 |
| 전체 저장 | `Cmd + Option + S` | 열린 파일 전체 저장 |

## TypeScript / NestJS 개발할 때 유용한 단축키

| 기능 | Mac 단축키 | 설명 |
|---|---:|---|
| 자동 import / Quick Fix | `Cmd + .` | `CreateDateColumn`, `Injectable` 같은 import 자동 추가 |
| 코드 포맷팅 | `Shift + Option + F` | Prettier 등으로 코드 정리 |
| import 정리 | `Shift + Option + O` | 안 쓰는 import 제거, import 정렬 |
| 정의로 이동 | `F12` 또는 `Cmd + Click` | 함수, 클래스, 타입 정의 위치로 이동 |
| 이전 위치로 돌아가기 | `Ctrl + -` | 정의로 이동한 뒤 원래 위치로 돌아가기 |
| 참조 찾기 | `Shift + F12` | 해당 함수/클래스가 사용된 곳 찾기 |
| 이름 한 번에 변경 | `F2` | 변수명, 클래스명, 함수명 전체 변경 |
| 에러 목록 보기 | `Cmd + Shift + M` | Problems 패널 열기 |

## 코드 편집 속도 올리는 단축키

| 기능 | Mac 단축키 | 설명 |
|---|---:|---|
| 한 줄 복사 | `Shift + Option + ↓ / ↑` | 현재 줄을 아래/위로 복사 |
| 한 줄 이동 | `Option + ↓ / ↑` | 현재 줄을 아래/위로 이동 |
| 한 줄 삭제 | `Cmd + Shift + K` | 현재 줄 삭제 |
| 주석 처리 | `Cmd + /` | 선택 영역 또는 현재 줄 주석 처리 |
| 같은 단어 추가 선택 | `Cmd + D` | 같은 단어를 하나씩 추가 선택 |
| 여러 커서 만들기 | `Option + Click` | 원하는 위치에 커서 여러 개 생성 |
| 현재 줄 선택 | `Cmd + L` | 현재 줄 전체 선택 |
| 파일 맨 위로 이동 | `Cmd + ↑` | 파일 최상단으로 이동 |
| 파일 맨 아래로 이동 | `Cmd + ↓` | 파일 최하단으로 이동 |

## 탐색 관련 단축키

| 기능 | Mac 단축키 | 설명 |
|---|---:|---|
| Explorer 열기 | `Cmd + Shift + E` | 파일 탐색기 패널 |
| Source Control 열기 | `Ctrl + Shift + G` | Git 변경사항 보기 |
| Extension 열기 | `Cmd + Shift + X` | 확장 프로그램 검색 |
| 현재 파일의 심볼 보기 | `Cmd + Shift + O` | 함수, 클래스, 변수 목록 |
| 프로젝트 전체 심볼 검색 | `Cmd + T` | 프로젝트 전체에서 클래스/함수 검색 |

## 디버깅 관련 단축키

| 기능 | Mac 단축키 | 설명 |
|---|---:|---|
| 디버깅 시작 | `F5` | Debug 실행 |
| 디버깅 없이 실행 | `Ctrl + F5` | 디버거 없이 실행 |
| 브레이크포인트 설정 | `F9` | 현재 줄에 중단점 설정 |
| Step Over | `F10` | 다음 줄 실행 |
| Step Into | `F11` | 함수 내부로 들어가기 |
| Step Out | `Shift + F11` | 함수 밖으로 나오기 |

- 코드 정리: `Shift + Option + F`  
- import 정리: `Shift + Option + O`  
- 파일 빠르게 열기: `Cmd + P`  
- 전체 검색: `Cmd + Shift + F`  
- 터미널 열기: <code>Ctrl + `</code>  
- 정의로 이동: `Cmd + Click`  
- 이름 한 번에 변경: `F2`
