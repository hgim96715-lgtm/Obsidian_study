
## 한 줄 요약

VS Code 확장 프로그램은 개발 생산성을 높여주는 도구이며, 너무 많이 설치하기보다 현재 공부하는 기술에 맞춰 필요한 것만 설치하는 것이 좋다.

---

# ⭐️ 공통 필수 확장

## Prettier

```txt
이름: Prettier - Code formatter
용도: 코드 자동 정렬
추천 이유: JavaScript, TypeScript, JSON, Markdown 포맷팅에 자주 사용
```

## ESLint

```txt
이름: ESLint
용도: JavaScript / TypeScript 코드 문제 감지
추천 이유: NestJS, Node.js 프로젝트에서 코드 품질 체크에 도움
```

## Material Icon Theme

```txt
이름: Material Icon Theme
용도: 파일/폴더 아이콘 보기 좋게 표시
추천 이유: 파일 구조 파악이 쉬워짐
```

## GitLens

```txt
이름: GitLens
용도: Git 변경 이력, 작성자, 커밋 추적
추천 이유: 협업 코드나 과거 변경 이유를 볼 때 유용
```

# ️ JavaScript / TypeScript / NestJS

## TypeScript Importer

```txt
용도: TypeScript import 자동완성 보조
```

## Error Lens

```txt
용도: 에러 메시지를 코드 라인 옆에 바로 표시
추천 이유: TypeScript 에러를 빠르게 확인 가능
너무 글이 많고 다 영어라서 불편..
```

## Jest

```txt
용도: Jest 테스트 실행 및 결과 확인
추천 이유: NestJS 단위 테스트 공부할 때 유용
```
