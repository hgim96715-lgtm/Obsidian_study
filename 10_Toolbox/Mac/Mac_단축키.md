## 1. Mac 단축키 기호 읽는 방법

| 기호 | 키 이름 | 역할 |
| --- | --- | --- |
| `⌘` | Command | 복사, 저장, 검색, 앱 전환 등 핵심 조작 |
| `⌥` | Option | 추가 기능, 단어 단위 이동, 파일 복사/이동 |
| `⌃` | Control | 시스템 기능, 우클릭, 화면 잠금 |
| `⇧` | Shift | 반대 방향, 범위 선택, 추가 동작 |
| `fn` / `🌐` | Function / 지구본 | 기능 키, 이모지, 특수 동작 |
| `⌫` | Delete | 커서 왼쪽 문자 삭제 |
| `esc` | Escape | 취소, 창 닫기, 실행 중단 |

### Windows와 비교해서 기억하기

| 하고 싶은 일 | Windows | Mac |
| --- | --- | --- |
| 복사 | `Ctrl + C` | `⌘ + C` |
| 붙여넣기 | `Ctrl + V` | `⌘ + V` |
| 저장 | `Ctrl + S` | `⌘ + S` |
| 찾기 | `Ctrl + F` | `⌘ + F` |
| 실행 취소 | `Ctrl + Z` | `⌘ + Z` |

## 2. 이것만 먼저 외우기 ⭐

| 단축키 | 기능 | 언제 쓰는지 |
| --- | --- | --- |
| `⌘ + Space` | Spotlight 검색 | 앱, 파일, 설정 빠르게 열기 |
| `⌘ + Tab` | 앱 전환 | VSCode ↔ Chrome ↔ DataGrip 이동 |
| `⌘ + C` | 복사 | 텍스트 또는 파일 복사 |
| `⌘ + V` | 붙여넣기 | 복사한 내용 붙여넣기 |
| `⌘ + Z` | 실행 취소 | 실수한 작업 되돌리기 |
| `⌘ + S` | 저장 | 문서, 코드 파일 저장 |
| `⌘ + F` | 찾기 | 문서·웹페이지·코드에서 검색 |
| `⌘ + W` | 현재 창 또는 탭 닫기 | 브라우저 탭, Finder 창 닫기 |
| `⌘ + Q` | 앱 종료 | 앱 자체를 완전히 종료 |

## 3. 앱과 창 관리

### 앱 전환

| 단축키                        | 기능                 |
| -------------------------- | ------------------ |
| `⌘ + Tab`                  | 열려 있는 앱 사이를 전환     |
| `⌘ + Tab` 누른 상태에서 `Tab` 반복 | 다음 앱으로 이동          |
| `⌘ + `` `                  | 같은 앱 안의 여러 창 사이 이동 |

### 창 닫기와 앱 종료의 차이

| 단축키 | 기능 | 차이 |
| --- | --- | --- |
| `⌘ + W` | 현재 창 또는 탭 닫기 | 앱은 계속 실행 중일 수 있음 |
| `⌘ + Q` | 현재 앱 종료 | 앱 자체가 종료됨 |
| `⌥ + ⌘ + Esc` | 앱 강제 종료 | 앱이 멈췄을 때 사용 |

### 화면 관리

| 단축키 | 기능 |
| --- | --- |
| `⌃ + ⌘ + F` | 전체 화면 시작 / 종료 |
| `⌃ + ↑` | Mission Control 열기 |
| `⌃ + ↓` | 현재 앱의 열린 창 모두 보기 |
| `⌘ + M` | 현재 창 최소화 |
| `⌃ + ⌘ + Q` | 화면 잠금 |

### 개발할 때 예시

```text
Chrome에서 공식 문서 확인
    ↓ ⌘ + Tab
VSCode에서 코드 수정
    ↓ ⌘ + Tab
Postman에서 API 요청
    ↓ ⌘ + Tab
DataGrip에서 DB 결과 확인
```

## 5. 스타일 없이 붙여넣기 ⭐

웹페이지나 문서에서 복사한 텍스트를 Obsidian에 붙여넣으면  
글꼴, 색상, 링크 스타일 등이 함께 들어올 때가 있다.

이럴 때는 일반 붙여넣기 대신 아래 단축키를 사용한다.

| 단축키 | 기능 |
| --- | --- |
| `⌥ + ⇧ + ⌘ + V` | 주변 문서 스타일에 맞춰 붙여넣기 |

### 활용 예시

```text
공식 문서에서 설명 복사
    ↓
Obsidian에 정리
    ↓
⌥ + ⇧ + ⌘ + V
    ↓
불필요한 서식 없이 깔끔하게 붙여넣기
```

## 6. Finder — 파일과 폴더 관리

Finder는 Mac에서 파일과 폴더를 관리하는 프로그램이다.

### 자주 쓰는 Finder 단축키

| 단축키 | 기능 | 활용 |
| --- | --- | --- |
| `⌘ + N` | 새 Finder 창 열기 | 새 파일 탐색 창 |
| `⇧ + ⌘ + N` | 새 폴더 생성 | 프로젝트 자료 폴더 만들기 |
| `Space` | 선택한 파일 미리보기 | 이미지, PDF 빠르게 확인 |
| `⌘ + I` | 파일 정보 보기 | 경로, 크기, 권한 확인 |
| `⌘ + Delete` | 선택한 파일을 휴지통으로 이동 | 파일 삭제 |
| `⇧ + ⌘ + Delete` | 휴지통 비우기 | 삭제 확정이므로 주의 |
| `⌥ + ⌘ + L` | 다운로드 폴더 열기 | 받은 파일 찾기 |
| `⇧ + ⌘ + H` | 홈 폴더 열기 | 사용자 기본 폴더 이동 |
| `⇧ + ⌘ + U` | 유틸리티 폴더 열기 | 터미널 등 확인 |
| `⇧ + ⌘ + G` | 특정 폴더 경로로 이동 | 개발 폴더 접근 |
| `⌥ + ⌘ + P` | 경로 막대 표시 / 숨기기 | 현재 폴더 경로 확인 |

---

## 7. 개발 공부할 때 특히 유용한 Finder 기능

### 7.1 특정 경로로 바로 이동하기

`⇧ + ⌘ + G`를 누르면 경로를 직접 입력해 폴더로 이동할 수 있다.

예를 들어 NestJS 프로젝트의 업로드 폴더를 확인하고 싶다면:

```text
/Users/내이름/프로젝트명/public/temp
```

경로를 입력하여 해당 폴더로 바로 이동할 수 있다.

### 활용 상황

```text
NestJS에서 파일 업로드 실행
    ↓
public/temp 경로에 파일이 실제 생성되었는지 확인
    ↓
Finder에서 ⇧ + ⌘ + G
    ↓
폴더 경로 입력 후 확인
```

---

### 7.2 파일을 열지 않고 미리보기

Finder에서 파일을 선택한 후 `Space`를 누르면  
앱을 실행하지 않고 파일 내용을 미리 볼 수 있다.

| 파일 종류 | 활용 |
| --- | --- |
| 이미지 | 업로드된 poster 파일 확인 |
| 영상 | 업로드한 mp4 파일 확인 |
| PDF | 강의 자료 빠르게 확인 |
| 텍스트 파일 | 내용 확인 |

```text
파일 선택 → Space → 미리보기 → Space 또는 esc로 닫기
```

## 8. 텍스트 작성과 커서 이동 ⭐

Obsidian, VSCode, 브라우저 입력창에서 모두 유용하게 사용할 수 있다.

### 커서 이동

| 단축키 | 기능 |
| --- | --- |
| `⌘ + ←` | 현재 줄의 맨 앞으로 이동 |
| `⌘ + →` | 현재 줄의 맨 뒤로 이동 |
| `⌘ + ↑` | 문서의 맨 위로 이동 |
| `⌘ + ↓` | 문서의 맨 아래로 이동 |
| `⌥ + ←` | 이전 단어로 이동 |
| `⌥ + →` | 다음 단어로 이동 |

### 텍스트 선택

| 단축키 | 기능 |
| --- | --- |
| `⇧ + ← / →` | 한 글자씩 선택 |
| `⌥ + ⇧ + ← / →` | 단어 단위로 선택 |
| `⇧ + ⌘ + ←` | 커서부터 현재 줄의 맨 앞까지 선택 |
| `⇧ + ⌘ + →` | 커서부터 현재 줄의 맨 뒤까지 선택 |
| `⇧ + ⌘ + ↑` | 커서부터 문서 맨 위까지 선택 |
| `⇧ + ⌘ + ↓` | 커서부터 문서 맨 아래까지 선택 |

### 삭제

| 단축키 | 기능 |
| --- | --- |
| `Delete` | 커서 왼쪽 문자 삭제 |
| `fn + Delete` | 커서 오른쪽 문자 삭제 |
| `⌥ + Delete` | 커서 왼쪽 단어 전체 삭제 |

### 코드 수정할 때 예시

```ts
const movieUpdateFields = {
    ...movieRest,
    director: newDirector,
};
```

`director` 단어 전체를 빠르게 지우고 싶을 때:

```text
director 뒤에 커서 위치
    ↓
⌥ + Delete
    ↓
단어 단위 삭제
```
